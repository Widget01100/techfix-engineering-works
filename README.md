# TechFix Engineering Works

A modern, responsive company website built with **HTML, CSS, and JavaScript**.  
This project is intended to showcase **TechFix Engineering Works** with a professional web presence.

---

## 🌐 Features
- Responsive landing page
- About, Contact, and 404 error pages
- Organized structure:
  - `/css` → Stylesheets
  - `/js` → JavaScript
  - `/images` → Project assets
- Nginx-ready for deployment
- Version controlled with Git & GitHub
